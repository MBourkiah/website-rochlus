# Rochlus — Logo & Favicon Paket

Abgeleitet aus dem Light-Editorial-Design-Konzept der Website (Fraunces/Instrument Sans, Fugen-Grid/Verband-Motiv, Terrakotta-Akzent).

## Dateien
- favicon.svg — Fugen-Grid-Icon (4 Fliesen), fuer website/public/favicon.svg
- icon-verband.svg — Bildmarke „versetzte Fliesen" (Alternative, fuer groessere Anwendungen)
- logo-lockup-light.svg — Wortmarke auf hellem Papier-Hintergrund
- logo-lockup-dark.svg — Wortmarke auf dunklem Kapitel-Hintergrund

## Einbindung
Ersetze `website/public/favicon.svg` durch `favicon.svg` aus diesem Paket. Fuer den Header-Logo-Slot eignet sich `logo-lockup-light.svg` (bzw. `-dark.svg` im dunklen Kontrast-Kapitel).

Hinweis: Die Wortmarken nutzen Fraunces/Instrument Sans als Web-Fonts — beim Export als PNG/Favicon-ICO ohne Fontladung ggf. Fallback-Schrift pruefen.
